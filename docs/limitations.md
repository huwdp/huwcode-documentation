# Limitations

List of features disabled in the Windows build:

- REPL user input through the of -t argument when running HuwCode.
- Functions
    - sleep
    - weekday
    - weekdayName
    - year
    - monthName
    - random
    - month
    - currentTime
    - clock
    - regexSearch
    - regexReplace
    - regexMatch
    - timeFormat
    - httpGet
    - httpPost
    - httpPut
    - httpDelete
    - now
